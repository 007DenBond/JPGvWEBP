# 🚀 DenBOND Creative Suite v5.4

**DenBOND Creative Suite** — это ультимативный инструмент для быстрой обработки медиаконтента, созданный специально для «вайб-кодинга» и автоматизации рутинных задач.

## 💎 Основные возможности

* **🖼 Фото:** Экспорт в выбранный формат (PNG / WEBP / JPG). Качество: WEBP 85, JPG 95.
* **🎬 Видео:** Рендер с водяным знаком через H.264 (`libx264`, `yuv420p`, `+faststart`, `preset medium`), аудио обрабатывается автоматически (AAC или без аудио).
* **🛡 Защита:** Четыре режима вотермарки:
    * **Справа внизу** — стандартный угол.
    * **Диагональ** — один логотип по центру (поворот 45°, плотность берется из ползунка).
    * **Сетка** — повтор 3×3 по кадру.
    * **Без вотермарки** — только экспорт фото/видео без наложения.
* **⚙ Интерфейс:** Ползунок называется **«Плотность вотермарки»**; выбор формата доступен только во вкладке **Фото**.

## 🛠 Технологии

* **Python 3**
* **CustomTkinter** — современный темный интерфейс.
* **MoviePy & Pillow** — мощные движки для обработки видео и фото.
