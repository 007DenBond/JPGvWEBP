# 🚀 DenBOND Creative Suite v5.3.1

**DenBOND Creative Suite** — это ультимативный инструмент для быстрой обработки медиаконтента, созданный специально для «вайб-кодинга» и автоматизации рутинных задач.

## 💎 Основные возможности

* **🖼 Фото:** Умное сжатие (WEBP/JPEG/PNG) с сохранением качества. Золотой стандарт — 85%.
* **🎬 Видео:** Рендер под Mac (MP4/MOV), изменение разрешения (1080p/720p) и функция Mute.
* **🛡 Защита:** Наложение вотермарков в трех режимах:
    * **В углу** — аккуратный брендинг.
    * **Диагональ** — стильная защита по центру.
    * **Сетка** — максимальная безопасность контента.

## 🛠 Технологии

* **Python 3**
* **CustomTkinter** — современный темный интерфейс.
* **MoviePy & Pillow** — мощные движки для обработки видео и фото.
