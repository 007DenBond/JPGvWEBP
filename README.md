# 🚀 DenBOND Creative Suite v5.3.1

**DenBOND Creative Suite** — это ультимативный инструмент для быстрой обработки медиаконтента, созданный специально для «вайб-кодинга» и автоматизации рутинных задач.

## 💎 Основные возможности

* **🖼 Фото:** Экспорт в том же формате, что и исходник (PNG → PNG, JPEG с качеством 95, WEBP с качеством 95).
* **🎬 Видео:** Рендер MP4/MOV (H.264, AAC, битрейт 5 Мбит/с, preset medium).
* **🛡 Защита:** Четыре режима вотермарки:
    * **Справа внизу** — стандартный угол.
    * **Диагональ** — один логотип по центру (прозрачность 30%, поворот 45°).
    * **Сетка** — повтор 3×3 по кадру.
    * **Без вотермарки** — только экспорт фото/видео без наложения.

## 🛠 Технологии

* **Python 3**
* **CustomTkinter** — современный темный интерфейс.
* **MoviePy & Pillow** — мощные движки для обработки видео и фото.
